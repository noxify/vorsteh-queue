This example demonstrates how to implement real-time progress tracking for long-running jobs, perfect for building progress bars and status indicators in your applications.

## Features Demonstrated

- **Real-time progress updates** - Jobs report progress as they execute
- **Multiple progress scenarios** - Dataset processing and file uploads
- **Progress event listening** - Monitor progress updates in real-time
- **Realistic simulations** - Chunked processing and upload scenarios

## Quick Start

Use the CLI to create this example:

```bash
npx create-vorsteh-queue my-app --template=progress-tracking
```

## Configuration

```bash
# Edit .env with your database URL
cp .env.example .env

# push the schema to the database
pnpm db:push
```

## Usage

```bash
# with pnpm
pnpm dev

# with npm
npm run dev
```
