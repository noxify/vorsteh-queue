---
title: Drizzle + postgres.js Example
---

Demonstrates Vorsteh Queue with Drizzle ORM and postgres.js driver. Shows job priorities, recurring jobs, and event handling.

## Setup

Use the CLI to create this example:

```bash
npx create-vorsteh-queue@latest my-project --template drizzle-postgres
cd my-project
cp .env.example .env
# Edit .env with your PostgreSQL database URL
pnpm db:push
pnpm dev
```
