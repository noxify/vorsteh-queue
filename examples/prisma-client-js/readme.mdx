---
title: Prisma Client JS Example
---

Demonstrates Vorsteh Queue with Prisma ORM using the traditional `prisma-client-js` provider for backward compatibility.

## Setup

Use the CLI to create this example:

```bash
npx create-vorsteh-queue@latest my-project --template prisma-client-js
cd my-project
cp .env.example .env
# Edit .env with your PostgreSQL database URL
pnpm prisma:generate
pnpm db:push
pnpm dev
```
