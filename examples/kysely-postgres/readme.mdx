## Setup

Use the CLI to create this example:

```bash
npx create-vorsteh-queue@latest my-project --template kysely-postgres
cd my-project
cp .env.example .env
# Edit .env with your PostgreSQL database URL
pnpm db:push
pnpm dev
```
